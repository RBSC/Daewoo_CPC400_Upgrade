This folder contains the BIOS image for the main ROM chip (U38) and the EPROM chip for the slot expander board for the following configuration:

Config_D: MSX2+, MSX BIOS and BASIC are in slot 0, disk BIOS is in slot 0.3, SUBROM and KANJI drivers are in slot 3.0, on-board RAM is disabled, external RAM is in expanded slot 3.2, FMPAC is in slot 3.1.

The folder also contains the ROM image for the FMPAC board.

Use any UV-erasable 27C512 EPROM or electrically-erasable Winbond W27C512 EEPROM chips.
