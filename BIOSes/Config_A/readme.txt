This folder contains the BIOS image for the main ROM chip (U38) for the following configuration:

Config_A: MSX2, all BIOSes are in slot 0, on-board RAM is in slot 0.2.

Use any UV-erasable 27C512 EPROM or electrically-erasable Winbond W27C512 EEPROM chip.
