These are original firmwares for the PAL chips that were originally installed on Daewoo motherboards.

Thanks to Peter Yoo and 3 other enthusiasts from South Korea (see the project description for their names).

The reverse assembly was performed by Meteor-M. PLD versions were created by Wierzbowsky [RBSC].
