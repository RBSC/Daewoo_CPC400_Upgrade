Daewoo CPC-400/400S MSX2/MSX2+ Conversion Project
Created and Published by RBSC in 2026
Special thanks to Max Vlasov [Meteor-M]
-------------------------------------------------

These folders contain Gerber files, KiCAD source files. ROMs, BOMs, 3D case files and firmwares for the following boards:

FMPAC Light board (with stereo output)
Slot Expander Board
EPROM/SLOT Daughterboard
RAM Expansion Board
Older RAM Expansion Board (!different firmware for the Slot Expander is required!)

The boards and firmwares were created by Wierzbowsky [RBSC] and Max Vlasov [Meteor-M].
