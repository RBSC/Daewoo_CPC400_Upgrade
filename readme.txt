Daewoo CPC-400/400S MSX2/MSX2+ Conversion Project
Created and Published by RBSC in 2026
Special thanks to Max Vlasov [Meteor-M]
-------------------------------------------------

The documentation for the project can be found in the DOCs folder of this repository. It consists of two PDF files. One file contains
information about the history of the project and how it was created, the second file contains instructions on how to make necessary
modifications to the hardware and what files to burn/flash into EPROM/GAL chips.
